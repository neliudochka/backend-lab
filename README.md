# Backend lab

## Installation

Clone this repository
```bash
git clone https://github.com/neliudochka/backend-lab.git
```

## Build

Move to the repo
```bash
// from the directory where you cloned repo  
cd   backend-lab
```

Now build docker images with docker-compose
 ```
 docker-compose build
``` 

## Run

and run
 ```
 docker-compose up
```
